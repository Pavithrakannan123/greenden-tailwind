GreenDen Website 🌿

A modern and responsive plant shop website built using HTML and CSS.
This project is designed to showcase indoor plants, gardening products, and eco-friendly website design concepts.

📌 Project Overview

GreenDen is a simple front-end website created using HTML and CSS.
The website includes a clean user interface, responsive layout, navigation menu, product sections, and contact information.

This project helps beginners practice:

HTML structure
CSS styling
Responsive web design
Flexbox/Grid layouts
Navigation bar creation
Product card design
